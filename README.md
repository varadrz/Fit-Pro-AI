# RestroBoom AI | Predictive Metabolic & Biomechanical Intelligence 

A premium, high-precision Health & Fitness intelligence platform. Built with a cinematic Apple-inspired design language, RestroBoom AI synthesizes natural language dietary intake and real-time biomechanical data into actionable performance insights.

##  Design & Impact Philosophy
- **High-Precision Minimalism**: A high-contrast, elite interface focusing on single-column clarity and predictive data visualization.
- **Cinematic Motion Design**: Integrated GSAP "Expo" and "Power4" easing for weighted, professional transitions and scroll-bound parallax.
- **Semantic SEO Optimized**: Built with an HTML5 semantic architecture and deep-indexed metadata for maximum search visibility and social impact.

## 🚀 Intelligent Lab Suites

### 1. Nutrition Lab (Metabolic Intelligence)
- **Metabolic Synthesis**: Advanced natural language logging transmuted into performance insights via a hybrid AI engine (Local Ollama + USDA FDC).
- **Amino Acid Balance**: High-precision protein deficiency detection with localized India-specific food heuristics.
- **Synthetic Analysis**: Dynamic metabolic energy and lipid profile visualization with static, accessible data labeling.

### 2. Restaurant Lab (Outlet Intelligence)
- **Real-Time Discovery**: Synchronized menu extraction for global chains (McDonald's, Subway, KFC) and authentic regional outlets.
- **Deep-Link Routing**: A modern SPA routing system that generates unique, shareable URIs for specific dishes.
- **Macro-Compliance Filtering**: Locate outlets by Net Carbs, High-Protein, or Plant-Based profiles.

### 3. Fitness Lab (Biomechanical Lab)
- **Neural Posture Tracking**: Real-time biometric feedback via the RestroBoom Vision engine (MediaPipe).
- **Performance Posture Engine**: Specialized tracking for Sagittal Squats, Structural Pushups, and Axial Rotation.

### 4. Integrated Ecosystem & Trust
- **Unified Brand Wall**: A cinematic, 3-row horizontal marquee featuring 24+ global health, fitness, and food brands.
- **Trust-Verified Intelligence**: Reorganized into targeted segments (Food Ecosystems, Delivery Partners, and Regulatory Licenses) to reflect a fully integrated health landscape.

## 🛠 High-Performance Stack
- **Frontend**: HTML5 (Semantic), Vanilla JS (ES Modules), Custom Glassmorphic CSS.
- **Motion**: GSAP 3.12 + ScrollTrigger (Weighted Cinematic Easing).
- **Intelligence**: Local Ollama (Ollama-CORS enabled) + MediaPipe Vision.
- **Data**: USDA FoodData Central API + TheMealDB.

## 🏁 Getting Started

### 1. Strategic Requirements
- **Node.js** (v18+)
- **Ollama**: Must be running locally with `ollama serve` and the `llava` model pulled.
- **Environment**: Create a `.env` file with your API credentials.

### 2. Initialization
```bash
# Clone and install dependencies
npm install

# Configure Environment Variables
# VITE_USDA_API_KEY=YOUR_KEY
# VITE_INDIAN_FOOD_API_TOKEN=YOUR_TOKEN

# Execute Laboratory
npm run dev
```

The application will be live at `http://localhost:3000/`.
